# 📈 Sales Forecasting Project

## 📌 Problem Statement
Predict future sales based on historical data.

## 📂 Dataset
Retail Sales Time Series Dataset

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Statsmodels (ARIMA)

## ⚙️ Model Used
- ARIMA (AutoRegressive Integrated Moving Average)

## 📊 Features
- Time series analysis
- Trend visualization
- Forecasting next 30 days

## 📈 Results
- Successfully predicted future sales
- Identified trends and seasonal patterns

## 🔍 Insights
- Sales show increasing trend over time
- Seasonal spikes observed in certain periods

## 🚀 Future Improvements
- Use Facebook Prophet
- Add external factors like promotions

## 📌 Conclusion
This project helps businesses plan inventory and strategy using future sales predictions.
